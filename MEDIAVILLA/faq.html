IV. Railroad Simulation

Q1: Is your own built route real or fictional?
A1: Purely fictional, except stations' names and othe locations.

Q2: "Signal stuck at red" on some parts
A2: Kindly press "TAB" to request SPAD.

If Approved:

  1. Ensure the AFB is switched "OFF".
  2. Drive at slow speed of 40 km/h or below when passing the signal.
  3. Press the "DELETE" key (PZB Override) to override emergency brake aplication.

If Denied:

  1. Check track switches if it is in the correct position.
  2. Ensure that the other train cleared its next block.

If Expired:

  1. Try to request SPAD again.
  2. Ensure that you passed the signal immediately.

Q3: "Cannot copy file/too large to destination file system"
A3: Possible solutions:

  1. Ensure that the drive(s) file system is not FAT32, as it limits file transfers up to 4GB.
  2. Use a NTFS or EXFAT drive(s).
  3. If you are using a flash drive to backup this route, kindly format it to NTFS or EXFAT.

Q4: "Archive is corrupt"
